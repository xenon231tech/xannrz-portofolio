## 🚀 Quick Start

### Step 1: Clone & Setup
```bash
git clone https://github.com/xenon231tech/xannrz-portofolio.git
cd xannrz-portofolio
npm install
```

### Step 2: Configure
```bash
cp .env.example .env.local
# Edit .env.local dengan konfigurasi Anda
```

### Step 3: Develop
```bash
npm run dev
# Buka http://localhost:3000
```

### Step 4: Deploy
```bash
npm run build
npm start
# Atau deploy ke Vercel
vercel deploy --prod
```

## 🎨 Customization

### Update Content
- Edit `src/data/` files dengan informasi Anda
- Update `src/components/` untuk styling
- Modifikasi `public/` untuk assets

### Change Theme
- Edit `tailwind.config.js` untuk warna
- Modifikasi `src/styles/globals.css` untuk font
- Update CSS variables di `src/styles/variables.css`

### Add Projects
- Buat project baru di `src/data/projects.json`
- Tambahkan project images ke `public/projects/`
- Update project component
