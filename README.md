```bash

git clone https://github.com/xannrz/xannrz-portofolio.git
cd xannrz-portofolio

npm install

npm run dev
```


## 🚀 Quick Start

### 1. Clone & Setup
```bash
git clone https://github.com/xannrz/xannrz-portofolio.git
cd xannrz-portofolio
npm install
```

### 2. Configure
```bash
cp .env.example .env.local
# Edit .env.local with your settings
```

### 3. Develop
```bash
npm run dev
# Open http://localhost:3000
```

### 4. Deploy
```bash
npm run build
npm start
# Or deploy to Vercel
vercel deploy --prod
```

## 🎨 Customization

### Update Content
- Edit `src/data/` files with your information
- Update `src/components/` for styling
- Modify `public/` for assets

### Change Theme
- Edit `tailwind.config.js` for colors
- Modify `src/styles/globals.css` for fonts
- Update CSS variables in `src/styles/variables.css`

### Add Projects
- Create new project in `src/data/projects.json`
- Add project images to `public/projects/`
- Update project component
